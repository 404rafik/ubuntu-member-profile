# Rafik Elouerchefani — Ubuntu Member Profile

This repository contains source files for a personal profile page, initially intended for [people.ubuntu.com](https://people.ubuntu.com), but forkable and deployable to any hosting service.

## About people.ubuntu.com

A web hosting service for recognized [Ubuntu Members](https://ubuntu.com/project/docs/community/membership/), offering a `public_html/` directory for professional profiles and project assets. See the official wiki for details.

## Deployment

1. Follow the [wiki guide](https://wiki.ubuntu.com/PeopleUbuntuCom)
2. Upload assets to `public_html/` — page becomes available at `http://people.ubuntu.com/~yourusername/`
